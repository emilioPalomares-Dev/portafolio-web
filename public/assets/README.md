# Portfolio Assets

This directory contains all media assets for the portfolio.

## Structure

- `resume/` - CV files in PDF format (English and Spanish)
- `drone.jpg` - Autonomous Drone project image
- `robotic-arms.jpg` - Robotic Arms Production Line project image
- `iot-access.jpg` - IoT Access Control System project image

## Update Instructions

Replace the placeholder images and CV files with your actual content:

1. Add your CV files:
   - `CV_EN.pdf` (English version)
   - `CV_ES.pdf` (Spanish version)

2. Add your project images:
   - Recommended size: 1200x800px
   - Format: JPG or WebP for optimal performance
   - Use descriptive filenames

3. Update the GitHub links in `src/data/projects.js` with your actual repositories
